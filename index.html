    Authentication

    Signup

    Request body

    {
  "email": "example@e.mail",
  "password": "password",
  "player_name": "player_name",
  "first_name": "first_name",
  "last_name": "last_name",
  "city": {
    "latitude": 0.0,
    "longitude": 0.0
  }
}

    Return 201 Created

    {}

    This endpoint create a player account for the user.

    HTTP Request

    POST {API_URL}/signup

    Body parameters

    Parameter Description Required
    email The email to use for registration x
    password The password to login with x
    player_name The name that will be used in the application x
    first_name The firstname of the user x
    last_name The lastname of the user x
    city The city of the user x

    Login

    Request body

    {
  "email": "example@e.mail",
  "password": "password"
}

    Return 200 OK

    {
  "access_token": "azecocapzie", 
  "refresh_token": "acencoaieno", 
  "at_expires": 1627048679, 
  "rt_expires": 1658498279, 
  "role": "Player", 
  "id": "player_id"
}

    This endpoint login a player and returns his credentials.

    HTTP Request

    POST {API_URL}/login

    Body parameters

    Parameter Description Required
    email The user's email x
    password The user's password x

    Player

    All these endpoints should be call with a header containing `Authorization: Bearer {access_token}`. The `access_token` is given by the `/login` endpoint.

    This endpoint create a player.

    You must be logged in as an administrator.

    HTTP Request

    POST {API_URL}/player

    Body parameters

    Parameter Description Required
    email The player's email x
    player_name The player's in-app name x
    first_name The player's first name x
    last_name The player's last name x
    city The player's city x

    Get Player

    Return 200 OK

    [
  {
    "id": "player_1",

    "player_name": "",
    "first_name": "",
    "last_name": "",
    "city": {
      "latitude": 0,
      "longitude": 0
    },
    "verified": true
  }
]

    This endpoint get a player.

    You must be logged in as an administrator or as the player.

    HTTP Request

    GET {API_URL}/player/:playerID

    URL Parameters

    Parameter Description
    playerID The id of the player to get

    Update Player

    Request body

    {
    "player_name":"player_name",
    "first_name":"first_name",
    "last_name":"last_name",
    "city": {
        "latitude": 0.0,
        "longitude": 0.0
    }
}

    Return 200 OK

    {}

    This endpoint update a player.

    You must be logged in as an administrator or as the player.

    HTTP Request

    PUT {API_URL}/player/:playerID

    URL Parameters

    Parameter Description
    playerID The id of the player to get

    Body parameters

    Parameter Description Required
    player_name The player's in-app name
    first_name The player's first name
    last_name The player's last name
    city The player's city

    Delete Player

    Return 200 OK

    {}

    This endpoint deletes a player.

    You must be logged in as an administrator or as the player.

    HTTP Request

    DELETE {API_URL}/player/:playerID

    URL Parameters

    Parameter Description
    playerID The id of the player to get

    Errors

    Error Code Meaning
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- You're not authorized to access this endpoint.
    404 Not Found -- The specified item could not be found.
    500 Internal Server Error -- We had a problem with our server. Try again later.
